# NeonTrack – Tela de Login

Projeto simples de **tela de login em HTML e CSS**, focado em visual moderno e identidade neon. Serve como base para estudos de front-end ou integração futura com back-end.

## Visão Geral

A página contém:
- Logo centralizada  
- Campo de e-mail  
- Campo de senha  
- Botão de login  
- Fontes personalizadas via Google Fonts  

Nenhum JavaScript foi usado. É estrutura e estilo, ponto.

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Google Fonts  
  - Krona One  
  - Momo Trust Display  
  - Stack Sans Text  

## Como Executar

1. Clone o repositório:
```
https://github.com/Marcojaco/Paainel-de-Login.git
```


2. Abra o arquivo `index.html` no navegador.

Simples assim.

## Observações

- O botão **ENTRAR** está dentro de um link (`<a>`) que redireciona para o YouTube.  
Isso é apenas demonstrativo.

- Não existe autenticação real. É apenas a interface.
